Path Finding Visualizer

A desktop application developed using wxWidgets to visualize some path finding algorithms written in C++, such as Dijkstra, BFS, A* Search, Greedy Best First Search & Bi-directional BFS.

Path Planning Visualizer

A desktop application developed using wxWidgets to visualize the path planning process, starting from defining environment geometry and robot geometry, until generating a path using the graph generated based on the type of decomposition method selected.

Image-based Sudoku Solver

This is an image-based sudoku solver that takes an image as input, and then uses OpenCV to find out the Sudoku and transform it into a 2D vector, and then solve it using backtracking algorithm and eventually display a solved sudoku.
