Backtracking

Introduction

  • Backtracking is actually a kind of brute-force solution.
  • DFS can be deemed as a kind of backtracking algorithm.
  • When using backtracking algorithm, there are 3 questions you need to think of:
    • What is your track?
    • What are the available choices for each track?
    • What is the termination condition?
  • The following is the framework of backtracking algorithm:
    •                         
all_tracks = [];
def backtrack(track)
{
    if termination condition met:
        all_tracks.append(track)
        return

    for choice in choices:
        add choice into track
        backtrack(track)
        remove choice from track
}
