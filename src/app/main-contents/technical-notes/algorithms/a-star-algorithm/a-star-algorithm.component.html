A* Search Algorithm

Introduction

  • A* Search Algorithm is another algorithm for finding the shortest path between two nodes in a graph.
  • This algorithm has an formula of f(n) = g(n) + h(n) , where:
    • f(n) : Travel cost for the node n
    • g(n) : Distance travelled from starting point to node n
    • h(n) : Heuristic approximation of the value of the node n
  • Hence, when exploring for shortest path, A* Search Algorithm will look at the node with lowest F value first.
  • Min Heap is also commonly used to lower the time complexity for finding the lowest F value among the nodes to be explored.
  • The Heurestic function plays an important role as it determines how fast it finds the path. The better the Heurestic function, the shorter time it needs to find the path.
  • An example of Heurestic function is finding the distance between current node and destination node.
